class pyxmolpp2.polymer.ChainSelection¶

Bases: pybind11_builtins.pybind11_object

Ordered list of chains.

Order of chains within single Frame matches their construction order. Chains from different frames are ordered as parent frames.

Frames are ordered by Frame.index. If two frames have identical index, order is undefined. Generally it’s a bad idea to have Atom/Residue/Chain selection with elements from two frames with same index.

Order is preserved across manipulations with ChainSelection
